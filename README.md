## 前端安装依赖
```sh
npm install
```

## 前端启动
```sh
npm run dev
```

## 后端FastAPI启动
```sh
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

## 后端Python库依赖
```sh
fastapi 后端api框架
uvicorn ASGI服务器
pydantic 数据验证与模型定义
splmodel 数据库ORM
ultralytics YOLO模型
pillow 图像处理
numpy 数值计算
pandas 数据分析
torch PyTorch深度学习框架
torchvision 图像预处理
python-multipart FastAPI处理文件上传
hashlib 哈希去重
albumentations 数据增强
```
